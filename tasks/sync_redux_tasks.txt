Покроковий план роботи з проєктом корзини інтернет-магазину та підключення Redux:

1. Встановити Redux та необхідні бібліотеки
Встанови redux, react-redux та @reduxjs/toolkit, якщо ще цього не зробив.
Переконайся, що в проєкті налаштований Redux Toolkit для спрощення роботи.

2. Налаштувати Redux Store
Створи папку redux у своєму проєкті.
Усередині неї додай файл store.js для налаштування Redux Store.
Підключи в store cartSlice, який буде обробляти стан корзини.

3. Створити cartSlice
Створи файл cartSlice.js у папці redux.
У ньому опиши:
Початковий стан корзини: масив товарів, загальна вартість.
Ред'юсери для обробки дій: додавання товару, видалення товару, зміна кількості.
Експортуй дії (actions) та ред'юсер.

4. Підключити Redux Store до застосунку
Обгорни кореневий компонент застосунку (App) у Provider з react-redux.
Передай store у Provider.

5. Зв’язати компоненти з Redux
Форма додавання товару
Підключи компонент AddProductForm до Redux.
Надсилай дані форми у сховище через action addProduct.
Список товарів у корзині
Підключи компонент CartItemList до Redux.
Читай стан корзини зі сховища через useSelector.
Надсилай дії (actions) для:
Збільшення кількості товару.
Зменшення кількості товару.
Видалення товару.
Загальна вартість корзини
Підключи компонент CartSummary до Redux.
Отримуй загальну вартість товарів зі стану сховища.
Розраховуй вартість на основі масиву товарів.

6. Реалізувати дії
Додавання товару
Оновлюй масив товарів, додаючи новий товар.
Якщо товар з таким id вже існує, збільшуй його кількість.
Видалення товару
Видаляй товар із масиву за його id.
Зміна кількості
Збільшуй або зменшуй quantity у товару, якщо це можливо (мінімум 1).

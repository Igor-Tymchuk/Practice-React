 1: Розробка головного макета і базових сторінок
Створити головний макет із хедером, який включає навігаційне меню:
Посилання: Головна, Про нас, Контакти, Блог, Профіль.
Використовувати Outlet для відображення вмісту на сторінках.
Створити базові сторінки:
Головна сторінка (/) — простий текст "Ласкаво просимо!".
Сторінка "Про нас" (/about) — опис додатку або команди.
Сторінка "Контакти" (/contacts) — будь-яка контактна інформація.
Налаштувати маршрути для цих сторінок у React Router.
(npm install react-router-dom)


 2: Реалізація блогу з вкладеними маршрутами
Створити маршрут /blog для відображення списку статей.
Додати фіктивний список статей із назвами та коротким описом.
Кожна стаття має посилання на сторінку /blog/:id.
Налаштувати вкладений маршрут /blog/:id для окремої статті:
Отримувати id статті з параметрів маршруту.
Відображати назву та вміст статті, завантажуючи дані з мокових об’єктів.
Додати заголовок "Blog" у верхній частині сторінок /blog та /blog/:id.
(https://jsonplaceholder.typicode.com/posts)


 3: Реалізація профілю з асинхронним завантаженням даних
Створити маршрут /profile для сторінки профілю користувача.
Завантажувати дані користувача асинхронно (моковий API або JSON-файл).
Відображати ім'я, електронну пошту та іншу інформацію після завантаження.
Додати індикатор завантаження для сторінки профілю, використовуючи Suspense.
Переконатися, що сторінка працює коректно під час повільного завантаження даних.
(https://jsonplaceholder.typicode.com/users)
